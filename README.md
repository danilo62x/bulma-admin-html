# Bulma Admin / HTML

> HTML puro, sem framework

Para quem só quer os arquivos HTML, CSS e JS prontos para colar em qualquer stack. Sem build, sem dependências — apenas o layout.

## 🧱 Stack

- HTML5
- CSS3
- JavaScript
- Bulma

## ✨ Recursos

- HTML, CSS, JS puros
- Sem build tools
- Use em qualquer backend
- Mesma identidade visual

## 🚀 Como rodar

Sem build: abra o `index.html` direto no navegador.

Se preferir servir por HTTP local:

```bash
npx serve .
```
